OWASP Top 10 Implementation & Mitigation

A01:2021

Broken Access Control

Restrictions on what authenticated users are allowed to do are not properly enforced.

Threat Scenario: Attacker manipulates URL parameters to access other users' order history
Mitigation: Implement proper RBAC, JWT validation, API gateway authorization
A02:2021

Cryptographic Failures

Failure to protect sensitive data through encryption and proper key management.

Threat Scenario: Credit card data stored in plaintext in database logs
Mitigation: AES-256 encryption, Cloud KMS, TLS 1.3, data tokenization
A03:2021

Injection

Untrusted data is sent to an interpreter as part of a command or query.

Threat Scenario: SQL injection through search functionality exposing customer data
Mitigation: Parameterized queries, ORM frameworks, input validation, WAF rules
A06:2021

Vulnerable Components

Using dependencies with known vulnerabilities.

Threat Scenario: Log4j vulnerability in Java microservices allowing RCE
Mitigation: SCA tools, dependency scanning, automated patching, SBOM
