
## Setup and Deployment

### Local Development

1. Clone or download this repository
2. Open `index.html` in a web browser
3. All resources are self-contained and will work offline

### GitHub Pages Deployment

1. Upload all files to a GitHub repository
2. Enable GitHub Pages in repository settings
3. Set source to "Deploy from a branch" and select main/master branch
4. The application will be available at `https://[username].github.io/[repository-name]`

### Custom Deployment

The application is static and can be deployed to any web hosting service:
- Netlify
- Vercel
- AWS S3 + CloudFront
- Azure Static Web Apps
- Google Cloud Storage
